# 🚀 Быстрый старт - Простая система

## ⚡ 3 минуты до первого жеста!

### Что это?

**Самый простой способ** добавить свои жесты и начать их использовать.

- ✅ Без сложных настроек
- ✅ Без обучения нейросети
- ✅ Работает сразу
- ✅ Легко добавлять новые жесты

---

## 📋 Шаг 1: Установка (1 минута)

```bash
cd signvoice_ai
pip install -r requirements.txt
```

**Готово!** Больше ничего не нужно.

---

## 🎓 Шаг 2: Добавьте свои жесты (2 минуты)

### Windows:

Дважды кликните: `signvoice_ai/launch_simple_trainer.bat`

### Linux/macOS:

```bash
cd signvoice_ai
python gesture_trainer_simple.py
```

### Что делать:

```
1. Нажмите: [▶ Старт камеры]

2. Нажмите: [➕ Добавить новый жест]
   Введите название: "Привет"

3. Нажмите: [🔴 Записать образец]
   Выполните движение "Привет" 5-10 раз
   Нажмите: [⏹ Стоп запись]

4. Повторите для других жестов:
   - "Круг" (рисуйте круг в воздухе)
   - "Махать" (машите рукой)
   - "Вверх" (движение руки вверх)

5. Готово! Жесты автоматически сохранены
```

---

## 🎯 Шаг 3: Используйте жесты (30 секунд)

### Windows:

Дважды кликните: `signvoice_ai/launch_simple_recognizer.bat`

### Linux/macOS:

```bash
cd signvoice_ai
python gesture_recognizer_simple.py
```

### Что делать:

```
1. Нажмите: [▶ Старт]

2. Выполняйте ваши жесты перед камерой

3. Программа будет:
   - Распознавать жесты
   - Озвучивать их названия
   - Показывать в истории

4. Готово! Просто показывайте жесты 🎉
```

## 💡 Советы

### Хорошие жесты:

- ✅ **Махать** - рука движется из стороны в сторону
- ✅ **Круг** - рисуете круг в воздухе
- ✅ **Вверх** - движение руки вверх
- ✅ **Вниз** - движение руки вниз
- ✅ **Привет** - открытая ладонь качается
- ✅ **Хлопок** - две руки сближаются

### Для лучшего распознавания:

- 📹 Хорошее освещение
- 📹 Рука полностью в кадре
- 📹 7-10 образцов на жест
- 📹 Выполняйте естественно
- 📹 Можно быстро или медленно - система адаптируется

## ❓ Вопросы и ответы

### Q: Сколько жестов можно добавить?

**A:** Сколько угодно! Но рекомендуется 5-10 для начала.

### Q: Можно ли удалить жест?

**A:** Пока нет. Но можно удалить файл `gesture_templates.pkl` и начать заново.

### Q: Работает с двумя руками?

**A:** Да! Система видит обе руки и использует их для распознавания.

### Q: Плохо распознает, что делать?

**A:** 
1. Запишите больше образцов (10-15)
2. Делайте жесты четче
3. Улучшите освещение
4. Уменьшите "Мин. уверенность" в настройках

### Q: Где хранятся мои жесты?

**A:** В файле `signvoice_ai/gesture_templates.pkl`

### Q: Можно ли поделиться жестами?

**A:** Да! Просто скопируйте файл `gesture_templates.pkl` другу.
