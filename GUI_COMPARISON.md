## 1. 🌟 Modern GUI (CustomTkinter) - РЕКОМЕНДУЕТСЯ

### Запуск
```bash

# или
signvoice_ai\launch_modern_gui.bat  # Windows
bash signvoice_ai/launch_modern_gui.sh  # Linux/macOS
```

- ✨ **CustomTkinter** - современная библиотека UI

- 📜 Красивая прокручиваемая история
- ⚙️ Современные слайдеры настроек

## 🎯 Когда использовать каждый интерфейс

### Modern GUI - Рекомендуется для:
- ✨ Демонстрации проекта
- 🎨 Когда важен внешний вид
- 📊 Анализа работы системы
- 🎓 Обучения и экспериментов
- 💼 Презентаций и выставок
- 👥 Работы с пользователями

### Standard GUI - Подходит для:
- 📱 Повседневной работы
- 🔧 Разработки и тестирования
- 📊 Сбора статистики
- ⚙️ Настройки параметров
- 🎯 Баланса функций и производительности

### Classic - Идеален для:
- 🚀 Быстрого тестирования
- ⚡ Минимального использования ресурсов
- 🔬 Отладки алгоритмов
- 📷 Проверки камеры
- 💻 Слабых компьютеров

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
# Для Classic и Standard GUI
pip install opencv-python mediapipe torch pyttsx3 numpy Pillow

# Дополнительно для Modern GUI
pip install customtkinter

# Или просто
cd signvoice_ai
pip install -r requirements.txt
```

### 2. Выбор и запуск

```bash
# Modern GUI (рекомендуется)
python signvoice_ai/main_modern_gui.py

# Standard GUI
python signvoice_ai/main_gui.py

# Classic
python signvoice_ai/main.py
```

## ❓ FAQ

**Q: Какой интерфейс самый быстрый?**
A: Classic (OpenCV) - минимальные накладные расходы.

**Q: Какой интерфейс самый красивый?**
A: Modern GUI (CustomTkinter) - профессиональный Material Design.

**Q: Можно ли использовать все три интерфейса?**
A: Да, все три полностью независимы и используют один backend.

**Q: Нужно ли устанавливать дополнительные зависимости?**
A: Для Modern GUI нужен CustomTkinter, для остальных - нет.

**Q: Какой интерфейс использовать для обучения модели?**
A: Скрипты обучения не зависят от интерфейса.
